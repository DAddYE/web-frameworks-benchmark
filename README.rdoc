Web frameworks benchmark.

Check the wiki for the {latest benchmark results}[https://github.com/DAddYE/web-frameworks-benchmark/wiki] or the master README[https://github.com/DAddYE/web-frameworks-benchmark/blob/master/README.rdoc] for details about the benchmarks.

== Usage

To setup gem for server:  './benchmark.rb setup'
To start and stop servers: './benchmark.rb start|restart|stop'.
Use './benchmark.rb run' against already booted servers.
