Web frameworks benchmark.

This branch utilizes before_filters, helpers, layouts, and sessions

Recent results for the rendering (higher is better):

  Sinatra 1.0.0     => 769.71 rps
  Padrino 0.9.10    => 717.64 rps
  Merb 1.1.0        => 537.53 rps
  Rails 3.0.beta3   => 231.13 rps
  Ramaze 2010.04.04 => 179.44 rps

rps = requests per second

== Usage

To start and stop servers: './benchmark.rb start|stop'.
Use './benchmark.rb run' against already booted servers.
