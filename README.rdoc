Web frameworks benchmark.

This branch utilizes before_filters, helpers, haml, controllers, layouts, and sessions

Recent results for the rendering (higher is better):

  Padrino 0.9.10    => 675.79 rps
  Sinatra 1.0.0     => 652.0 rps
  Merb 1.1.0        => 642.29 rps
  Rails 3.0.beta3   => 201.86 rps
  Ramaze 2010.04.04 => 130.62 rps

rps = requests per second

== Usage

To start and stop servers: './benchmark.rb start|stop'.
Use './benchmark.rb run' against already booted servers.
